# NotebookLM Knowledge Pack: Instore Digitization

This folder contains a complete knowledge extraction of the `instore_digitization` repository, optimized for NotebookLM.

## Contents
- 11 `.docx` files covering Architecture, Workflows, Jira Tickets, and Requirements.
- `manifest.json` mapping sources to outputs.
